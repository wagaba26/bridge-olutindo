# Content Flow Summary

## Core progression
1. Homepage -> Defines the operating model (assess, train, certify) and sets expectation for measurable progression.
2. Japan Preparation Hub -> Expands guidance into three applied modules: working, studying, and living in Japan.
3. Intake -> Captures learner baseline, pathway intent, and support-desk routing for actionable follow-up.
4. Dashboard -> Executes learning plans with immersion controls, skill breakdown, and live teacher updates.

## Language usage model
- Public pages are English-first with Japanese terms in romaji for credibility and readability.
- Learning progression aligns with immersion stages:
  - N5: English-supported
  - N4: balanced bilingual
  - N3: Japanese-dominant
- Dashboard immersion mode now actively controls displayed immersion spectrum.

## Tone model
- Calm and structured.
- Measurable wording (hours, levels, sequence, milestones).
- No hype, no exaggerated promises, no guarantee language.
